# Book Tracker

## Автор
Мельников Роман

## Описание
Book Tracker — это GUI-приложение для учёта прочитанных книг.
Позволяет добавлять книги, фильтровать их и сохранять данные в JSON.

## Функции
- Добавление книги (название, автор, жанр, страницы)
- Фильтрация по жанру
- Фильтрация по количеству страниц
- Сохранение и загрузка данных (JSON)
