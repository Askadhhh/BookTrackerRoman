## Описание программы

Book Tracker — это GUI-приложение для учёта прочитанных книг.

Программа позволяет:

- добавлять книги;
- хранить список прочитанных книг;
- фильтровать книги по жанру;
- фильтровать книги по количеству страниц;
- сохранять данные в JSON-файл;
- загружать данные из JSON-файла.

---

## Основные функции

### Добавление книги

Пользователь может добавить:

- название книги;
- автора;
- жанр;
- количество страниц.

После нажатия кнопки «Добавить книгу» информация появляется в таблице.

### Фильтрация

Доступна фильтрация:

- по жанру;
- по количеству страниц (например, больше 200).

### Работа с JSON

Программа поддерживает:

- сохранение списка книг;
- загрузку списка книг при запуске программы.
